Beep Boop

Beep Booper #1

This is the assignment for the prompt. The submit button will count up to the number specified by the user and replace all ones with 'boop' and zeros with 'beep'. If the user enters a number that is divisible by 3 or a non-numeric entry, they will be notified that they need to choose a different number. If the number has a one in it, it will override any other zeros and show "boop".

Beep Booper #2

If the number has a one in it, it will override any other zeros and show "boop".
